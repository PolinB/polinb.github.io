Сравнение Playwright и Cypress

Плюсы и минусы Playwright:

  • + Запускается через jest, а значит можно запускать вместе с группой тестов
  • + Приятная документация
  • + Можно тестировать на разных браузерах
  • — Каждый раз приходится писать async и await

Плюсы и минусы Cypress:

  • + Удобный ui при запуске тестов через cypress open
  • + Можно тестировать на разных браузерах
  • + Приятная документация
  • — Запускается отдельно

Лично мне больше понравился cypress, удобно, просто, красиво, необходимая функциональность есть.

Тестирование с помощью скриншотов и видео

С помощью e2e тестов мы можем тестировать поведение пользователя и ожидаемые результаты.
Однако не все проблемы могут выявлены с помощью сравнений значений элементов.
Иногда может пригодится тестирование с записями скриншотов или видео.
Как мне кажется, это необходимо для ситуаций, когда возникают проблемы с css.
В своем проекте я такого не нашла, но на других сайтах замечала пробелмы с заезжанием текстов или с вылетающими списками.
